# Mockups

This directory stores Figma export files and design assets.

## Usage

1. Export designs from Figma (SVG, PNG, or JSON format)
2. Organize by feature or screen
3. Include design tokens if available
4. Document any special considerations

## Recommended Structure

```
mockups/
├── screens/          # Full screen mockups
├── components/       # Individual component designs
├── assets/           # Images, icons, illustrations
└── tokens/           # Design tokens (colors, spacing, typography)
```
